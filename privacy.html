Privacy Policy

Last updated: December 2025

PixUp ek browser-based image compression tool hai. Ye page batata hai ki hum is website par kaun se basic data handle karte hain aur aapke images ke sath kya hota hai.

1. Images ka processing

  • By default, saari image compression aapke browser ke andar hi hoti hai (client-side).
  • Iska matlab hai ki aapke images hamare server par auto-upload nahi hote jab tak aapne optional server conversion enable nahi kiya ho.

2. Server conversion (optional)

Agar aap advanced formats ke liye optional backend server use karte hain, to:

  • Image sirf conversion ke time temporary process hota hai.
  • Hum images ko analytics ya training ke liye store nahi rakhne ka intent rakhte hain.

3. Basic usage data

Is demo version me hum:

  • Koi login system ya user accounts use nahi karte.
  • Third-party trackers add nahi kiye gaye (jaise analytics scripts), jab tak aap khud add na karein.

4. Local storage

Browser kabhi-kabhi aapke recent downloads, files choose karne ke dialogs, ya theme preferences ko remember kar sakta hai. Ye behaviour browser ka hota hai, specifically PixUp ka nahi.

5. Contact

Agar aap is privacy text ko customize karna chahte hain (for production use), to apne legal advisor se consult karke yaha apne conditions likh sakte hain.
